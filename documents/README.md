## Dig ORM User Documentation

Welcome to **Dig**, an ORM (Object-Relational Mapping) library written in Zig.

This directory contains user‑facing documentation for Dig.
If you are *using* Dig in your own Zig application, start here.

- **Overview**: [`overview.md`](./overview.md)
  High‑level overview, design goals, and technical requirements.
- **Getting Started**: [`getting-started.md`](./getting-started.md)
  Install Zig, set up the project, and connect to your first database.
- **Schema Definition**: [`schema.md`](./schema.md)
  Define tables, columns, and generate CREATE TABLE statements.
- **Query Builders**: [`query-builders.md`](./query-builders.md)
  Build SELECT, INSERT, UPDATE, and DELETE queries with a fluent API.
- **Migrations**: [`migrations.md`](./migrations.md)
  Manage database schema versioning with SQL-based migrations.
- **Database Drivers**: [`database-drivers.md`](./database-drivers.md)
  PostgreSQL and MySQL driver details, type mappings, and integration.
- **API Reference (High‑Level)**: [`api-reference.md`](./api-reference.md)
  Summary of types, methods, and core components.
- **Architecture**: [`architecture.md`](./architecture.md)
  Detailed architecture documentation for contributors and advanced users.
